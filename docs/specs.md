## 概要
Android 版 LINE アプリを自動で操作して、アルバムに含まれる写真・動画のタイムスタンプの一覧を作成する

## 背景
LINEのアルバムにアップロードされた画像をダウンロードしたい。
撮影日時はLINEアルバムからは確認できるが、画像のEXIF情報は削除されてしまう。
そのため、画像をダウンロードしたあとにLINEアルバムを操作して撮影日時を確認する必要がある。
この操作を自動化したい。


## 必要な操作
- 画面上に現れる画像の index, (x/n) のうち x をダンプする。
  - この情報を i とおく
- 「再生」ボタンがあるかどうか判定し、あれば mp4, なければ jpg と判定する
  - ty に格納する
- 画面右上のメニューを開く
- 「詳細情報」をタップする
- 画面下に現れる情報のうち、「作成日時」「作成日時」をダンプする
  - この情報をそれぞれ u, ts とおく
- csv に `i, ty, u, ts` の行を追記する
- 画面を左にスワイプして次の画像を開く
- 画面上に現れる画像の index が (n/n) まで到達したら終了


## 最終的な成果物
画像、動画の index と registrant, timestamp を csv として export する


format:

```
index, filetype, timestamp, registrant
1, jpg, 2021/03/26 13:54, Kumassy
2, mp4, 2021/03/26 14:02, Kumassy
```
